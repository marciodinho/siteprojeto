The Game

CyberFurry is a "..." top-down 3D game for PC inspired by the CyberPunk 2077 style and the game Ratchet & Clank. The game features a storyline set in a dystopian future in the violent town of FurryTown where Z, the protagonist FurryPunk lives.
Developed with the game engine Unity, aimed at young audiences with ratings for all ages.
